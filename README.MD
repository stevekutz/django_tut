### Django tutorial

1) Set up virtual environment
    ~~~ bash
    python3 -m venv tut_env
    ~~~
2) Launch virtual environment
    ~~~ bash
    source tut_env/bin/activate
    ~~~
3) Install `Django` 
    ~~~ bash
    pip install django
    ~~~
- the version can be verified by running `python3 -m django --version`
